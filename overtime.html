OVERTIME REQUEST APP

This is a website that my team and I created during the training period at PT Mitra Integrasi Informatika.

This website is used to create overtime forms for employees. In this website there are three users, namely employees, managers and finance. Employees can do overtime forms. Meanwhile, the manager and finance confirm agree and reject the employee overtime form

The first step is that employees must log in and create an overtime form first, then the manager will approve or reject the user form, if the form is approved, proceed to finance if finance approves the overtime form, the employee gets overtime pay.

This website was created using the ASP.Net Core framework with the C# programming language.
